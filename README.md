# Rukhma Shafqat - Premium Portfolio Website

A world-class premium portfolio website for Rukhma Shafqat - Website Developer, Shopify Expert, and AI Automation Specialist.

## Features

✨ **Modern & Elegant Design** - Inspired by Apple, Linear, Framer, and Stripe
🎨 **Glassmorphism UI** - Beautiful glass effect cards and components
✨ **Premium Animations** - Smooth transitions and micro-interactions with Framer Motion
🌙 **Dark Mode** - Seamless dark mode toggle
📱 **Fully Responsive** - Works perfectly on all devices
⚡ **Fast Performance** - Built with Next.js for optimal speed
🔍 **SEO Optimized** - Complete SEO setup with schema markup
♿ **Accessible** - WCAG compliant

## Tech Stack

- **Framework**: Next.js 14
- **Language**: TypeScript
- **Styling**: Tailwind CSS
- **Animations**: Framer Motion & GSAP
- **Icons**: Lucide React
- **State Management**: Zustand
- **Scroll Effects**: React Scroll & Intersection Observer

## Getting Started

### Prerequisites

- Node.js 18+
- npm or yarn

### Installation

```bash
git clone https://github.com/sahiraijaz0-source/portfolio-website.git
cd portfolio-website
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) in your browser.
